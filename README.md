# LLM Component Generator

Generate React components from multiple LLMs and compare their outputs to choose your favorite.

## Features

- **Multi-LLM Support**: Compare GPT-4o, Claude Sonnet 4, Gemini 2 Pro, GLM-4.7, Kimi K2.5, MiniMax M2.1, DeepSeek V3, and Llama 4 Scout
- **Clean UI**: Minimalist design inspired by OpenRouter and Eleven Labs
- **Code & Preview**: View generated code and live previews side-by-side
- **One-Click Copy**: Easily copy the code you want to use

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Usage

1. Describe the React component you want
2. Select which LLMs to compare
3. Click "Generate Components"
4. Review code and previews
5. Copy your favorite

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide Icons
